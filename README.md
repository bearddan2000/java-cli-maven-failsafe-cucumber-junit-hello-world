# java-cli-maven-failsafe-cucumber-junit-hello-world

## Description
A POC for maven app using JUnit5
and cucumber framework with surefire
and failsafe plugins.

## Tech stack
- java
- maven
  - junit
  - failsafe
  - surefire
  - cucumber

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
